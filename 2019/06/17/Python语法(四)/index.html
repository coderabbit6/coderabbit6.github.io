检查是否相等

比较数字

  • 用<,>,<=,>=来判断

检查多个条件

  • 用and，or来连接两个或者多个条件

判断特定的值是否已包含在列表中

  • 用关键字in来判断在列表中

  • 用关键字not in来判断不在列表中
